## Prerequisites

<LinkButton to="connect-to-rover-ap" description="Connect to Leo Rover AP" />

With the steps provided, you'll be able to access a clean video stream preview in Leo Rover network.
The preview can be used to display the stream on a different screen or to be caught for further development.
It's up to you.

## Access the stream via your browser

When connected to Leo Rover's access point, enter the dedicated port.

Type in your browser:

```
http://10.0.0.1:8080/stream?topic=/camera/image_raw&type=ros_compressed
```
